Dependency injection container

Opis Container is a library that provides a fully serializable dependency injection container. The Opis Container library can be used to deliver a more reusable, testable and maintainable code, while increasing the speed of development.

License

Requirements

  • PHP 5.3.* or higher
  • Opis Closure ^2.0.0

Installation

Opis Container is available on Packagist and can be installed using Composer. 

{
    "require": {
        "opis/container": "^2.3.1"
    }
}

If you are unable to use Composer you can download the tar.gz or the zip archive file, extract the content of the archive and include de autoload.php file into your project. 

require_once 'path/to/container-2.3.1/autoload.php';
