Introduction

Responding to an HTTP request is done by creating a new instance of the Opis\Http\Response class by passing as an argument to the constructor of the class, an instance of the Opis\Http\Request class, representing the current HTTP request. 

use Opis\Http\Request;
use Opis\Http\Response;

$request = Request::fromGlobals();
$response = new Response($request);

Another way of obtaining a Response object is by using the response method on a Request object. 

use Opis\Http\Request;

$request = Request::fromGlobals();
$response = $request->response();

Response content

Setting the content of the response is done with the help of the body method. The method accepts as an argument a string or an object that implements the magic method __toString. 

$response->body($content);

Setting the content type of the response can be done by using the contentType method. 

$response->contentType('text/html');

You can also specify a charset for the content type by passing a second argument to the contentType method, or by using the charset method. 

$response->contentType('text/html', 'utf-8');
//..
$response->charset('utf-8');

Getting the content that was previously set is done by using the getBody method. 

$content = $response->getBody();

You can also pass as an argument to the body method an anonymous function and print yourself the content of the response. The anonymous function will receive as arguments a Request and a Response object. 

$response->body(function($request, $response){
    echo "Content";
});

If you want to take the entire control over how a response is send back to the client, you can pass as an argument for the body method an object implementing the Opis\Http\HttpResponseInterface. 

namespace Opis\Http;

interface HttpResponseInterface
{
    public function handle(Request $request, Response $response);
}

Response headers

Setting a response header is done by using the header method. The method takes as an argument the header name and the header's value. 

$response->header('X-Foo', 'bar');

You can set multiple headers at once by using the headers method. 

$response->headers(array(
    'X-Foo' => 'bar',
    'X-Bar' => 'baz',
));

Clearing headers is done by using the clearHeaders method. 

$response->clearHeaders();

Sending the response headers is done by using the sendHeaders method. 

$response->sendHeaders();

Cookies

Setting a cookie is done with the help of the cookie method. The method takes as arguments the name of the cookie, the cookie's value, an optional value that specify when the cookie should expire and an optional array of settings. 

//Set a cookie `foo` containing a value `bar` that will expire when browser is closed.
$response->cookie('foo', 'bar');
// Set a cookie that will expire in an hour
$response->cookie('foo', 'bar', 3600);
// Setting cookie options
$response->cookie('foo', 'bar', 3600, array(
    'path' => '/baz/', 
    'domain' => 'exmaple.com',
    'secure' => false,
    'httponly' => false,
));

You can delete a cookie by using the deleteCookie method. The method takes as arguments the cookie name and an optional array of cookie options. 

// Delete a cookie `foo` 
$response->deleteCookie('foo');
// Delete a cookie `foo` that was setted using these options
$response->deleteCookie('foo', array(
    'path' => '/baz/', 
    'domain' => 'exmaple.com',
    'secure' => false,
    'httponly' => false,
));

Clearing all cookies is done by using the clearCookies method. 

$response->clearCookies();

Sending the response

Sending the response is done with the help of send method. The method will take care of sending all the headers and all the cookies that were set, along with the specified content. 

use Opis\Http\Request;

$request = Request::fromGlobals();
$response = $request->response();

$response->body('Hello world!');
$response->contentType('text/html');
$response->send();

You can specify a status code for a response by using the status method. 

use Opis\Http\Request;

$request = Request::fromGlobals();
$response = $request->response();

$response->body('Page not found');
$response->status(404);
$response->contentType('text/html');
$response->send();

Sending a redirect response is done with the help of the redirect method. 

use Opis\Http\Request;

$request = Request::fromGlobals();
$response = $request->response();

$response->redirect('/other/path');

You can also specify a redirect code (default is 302). 

$response->redirect('/other/path', 301);
