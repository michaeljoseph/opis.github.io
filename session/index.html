Session manager

Opis Session is a session manager library with support for multiple backend storages that provides developers with an API which allows them to handle session related informations in a standardised way.

The currently supported storages are: Database, File, Mongo and Redis.

{% include subtitle.html value="Requirements" %}
  • PHP 5.3.* or higher
  • Opis Database 2.0.* (for Database storage)
  • Predis 1.0.* (for Redis storage)
