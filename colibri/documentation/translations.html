Translations

  1. Introduction
  2. Adding new translations
  3. Working with translations

Introduction

The framework provides a translation system that allows users to perform translations from english language to any other language. Translations are collected using the Translations method. 

use Opis\Colibri\Define;

Define::Translations(function($translate){

    // Collect translations

});

You can override a translation by specifying a lower collecting priority. 

use Opis\Colibri\Define;

Define::Translations(function($translate){

    // Collect translations

}, -1);

The collector passed to the callback function is an object that implements the Opis\Colibri\TranslationCollectorInterface interface. 

interface TranslationCollectorInterface
{
    /**
     * Set the current used language
     *
     * @param   string  $language   Language
     *
     * @return  \Opis\Colibri\TranslationCollectorInterface   Self reference 
     */
    
    public function using($language);
    
    /**
     * Add the sentences that will be translated from english to current used language
     *
     * @param   array   $sentences  Trnslated sentences
     */
    
    public function translate(array $sentences);
}

Adding new translations

You can add a new translation by passing as an argument an array containing translations, from english to a previously chosen language, to the translate method. Choosing the translation language is done with the help of the using method. 

use Opis\Colibri\Define;

Define::Translations(function($translation){

    $translation->using('de')
                ->translate(array(
                    'Hello' => 'Hallo',
                ));
    
    $translation->using('fr')
                ->translate(array(
                    'Hello' => 'Bonjour',
                ));
    
});

Working with translations

Translating a text is done by passing that text as an argument to the T global function. 

echo T('Hello');

You can specify a translation language, by passing the language identifier as the second argument to the T function. 

echo T('Hello', 'de');

Setting a default translation language can be done by using the instance and setLanguage methods of the Opis\Colibri\Translator class.

Important! The system module contains some useful bindings that will automate the process of setting a default language, by taking into consideration the URL that was accessed. 

\Opis\Colibri\Translator::instance()->setLanguage('fr');
echo T('Hello');      // translate form en to fr
echo T('Hello', 'de') // translate form en to de
