Cache storages

  1. Introduction
  2. Registering cache storages
  3. Working with cached content

Introduction

The framework's cache support is provided by the Opis Cache library and cache storages are collected using the CacheStorages method. 

use Opis\Colibri\Define;

Define::CacheStorages(function($storage){

    // Collect cache storages

});

You can override a cache storage by specifying a lower collecting priority. 

use Opis\Colibri\Define;

Define::CacheStorages(function($storage){

    // Collect cache storages

}, -1);

The collector passed to the callback function is an object that implements the Opis\Colibri\CacheCollectorInterface interface. 

interface StorageCollectorInterface
{
    /**
     * Register a new storage
     *
     * @param   string      $storage        Storage name
     * @param   \Closure    $constructor    Storage constructor callback
     * @param   boolean     $default        (optional) Default flag
     *
     * @return  mixed
     */
    
    public function register($storage, Closure $constructor, $default = false);
}


interface CacheCollectorInterface extends StorageCollectorInterface
{
    
}

Registering cache storages

You can register a new cache storage by passing the storage name and an anonymous callback function to the register method. The callback must return an instance of a class that implements the Opis\Cache\StorageInterface interface. 

use Opis\Colibri\Define;
use Opis\Cache\Storage\File as FileStorage;

Define::CacheStorages(function($storage){

    $storage->register('foo', function(){
        return new FileStorage('/path/to/folder');
    });

});

Working with cached content

Working with cached content is done by using the Cache global function. This function takes as an argument the name of the cache storage you want to work with and returns an instance of the Opis\Cache\Cache class, that can be further used to manage cached content. 

Cache('foo')->write('bar', 'baz');

If the storage name is ommited, then the default storage will be used.

Important! The default cache storage is registred during the installation of the framework and can not be overriden. 

Cache()->write('bar', 'baz');
