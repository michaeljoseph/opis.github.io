HttpRequest

This function returns the underlying HTTP request object which is an instance of Opis\Http\Request. 

//Get a value from the querystring
$name = HttpRequest()->get('name');

HttpResponse

Returns the underlying HTTP response object represented by an instance of the Opis\Http\Response class. 

//Set the response's content type to 'image/jpeg'
$name = HttpResponse()->type('image/jpeg');

HttpRedirect

Performs an HTTP redirection to a specified location. This function takes as arguments the location and optionally a redirection code and an array containing query parameters. The default redirection code, if none was specified, is 302(302 Found). 

//Performs a '302 Found' redirect to http://example.com
HttpRedirect('http://example.com');

//Performs a '301 Moved Permanently' redirect to /test
HttpRedirect('/test', 301);

//Performs a '302 Found' redirect to /test?name=opis
HttpRedirect('/test', 302, array(
    'name' => 'opis',
));

GetURL

Returns the full URL of a given path. 

$url = GetURL('/some/path');

If you want that only the URL's full path to be returned, then you must pass false as the second argument to the function. 

$url = GetURL('/some/path', false);

PathFromRoute

Returns the path associated with a named route. The function takes as arguments the route's name and an array of values used to replace path's named segments. 

echo PathFromRoute('foo');

/*
 A route named `user-profile`, that have an `/profile/{username}` associated path ,
 will be mapped to `/profile/foo`
*/

echo PathFromRoute('user-profile', array(
    'username' => 'foo'
));

Asset

Returns the full URL of a module's asset. The function takes as arguments the module's name and the asset's path relative to the assets folder. 

// Outputs something like `http://www.example.com/assets/module/system/img/opis.png`
echo Asset('system', 'img/opis.png');

If you want that only the URL's full path to be returned, then pass false as the third argument to the function. 

// Outputs `/assets/module/system/img/opis.png`
echo Asset('system', 'img/opis.png', true);

View

Creates and return a new instance of the Opis\Colibri\View class. The function takes as arguments the view's name and an optional array of values that will be used by the view. 

echo View('foo.bar');

echo View('user.profile', array(
    'username' => 'foo',
    'age' => 21,
    'email' => 'foo@example.com',
));

Render

Renders an object that implements the Opis\View\ViewableInterface. 

echo Render($someObject);

echo Render(View('foo.bar'));

Emit

Creates and dispatch an event. The created event is an instance of the Opis\Colibri\Event class. The function takes as an argument the event's name and returns the dispatched event. 

$emitted = Emit('foo');

If you want that the created event to be cancelable, then you must pass true as the second argument of the function. 

$emitted = Emit('foo', true);

Dispatch

Dispatch an event. The event must be an instance of Opis\Colibri\Event class, or must be an object that inherits that class. The function returns the dispatched object. 

use Opis\Colibri\Event;

$event = Dispatch(new Event('foo'));

Using

Resolves an abstract type that was bound to a concrete type during the collecting stage. The function takes as arguments the abstract type's name and optionally an array of values that will be used as arguments when the abstract type is resolved. 

$obj1 = Using('SomeType');

$obj2 = Using('OtherType', array('foo', 'bar'));

Database

Returns an instance of Opis\Database\Database , that can be used to perform SQL specific operations. 

$results = Database()->from('foo')
                     ->select()
                     ->all();

Database('other-connection')->update('bar')
                            ->where('foo')->is('baz')
                            ->set(array(
                                'foo' => 'qux',
                            ));

Schema

Returns an instance of Opis\Database\Schema , that can be used to manage SQL tables. 

Schema()->create('foo', function($table){
  // Add fields
});

Schema('other-connection')->drop('bar');

Session

Returns an instance of Opis\Session\Session , that can be used to manage users' sessions. 

Session()->set('authenticated', true);

Cache

Returns an instance of Opis\Cache\Cache , that can be used to manage cached content. 

Cache()->write('foo', 'bar');

Cache('other-storage')->delete('baz');

Config

Returns an instance of Opis\Config\Config , that can be used to manage configurations. 

Config()->write('foo', 'bar');

Config('other-storage')->delete('baz');

Module

Returns an instance of Opis\Colibri\ModuleInfo , that can be used to obtain various information about a module. The function takes as an argument the module's name. 

$module = Module('system');

if($module->isInstalled())
{
    // Do something
}

T function

Translates a given text. Optionally you can pass the translation language as the second argument to the function. 

echo T('Hello');

echo T('Hello', 'fr');

Variable

Returns the value of a stored variable. The function takes as an argument the variable's name. You can also specify default value to be returned, if the variable doesn't exist, by passing that value as a second argument to the function. 

$foo = Variable('foo');

$foo = Variable('foo', 'bar');

CSRFToken

Generates a CSRF token. 

$token = CSRFToken();

CSRFValidate

Checks if the given value is a valid CSRF token. 

if(CSRFValidate($token))
{
    // Do something
}
