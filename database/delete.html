Delete records

Deleting records is done by using the from and delete methods.

{% highlight php startinline %} // Delete all users $result = $db->from('users') ->delete(); {% endhighlight %} {% highlight sql %} DELETE FROM `users` {% endhighlight %}

Deleting a specific set of records is done by adding filtering conditions.

{% highlight php startinline %} // Delete all users which don't have a description $result = $db->from('users') ->where('description')->isNull() ->delete(); {% endhighlight %} {% highlight sql %} DELETE FROM `users` WHERE `description` IS NULL {% endhighlight %}

You can also delete from multiple tables simultaneously by performing a join and passing to the delete method a list of tables as an array argument.

{% highlight php startinline %} // Delete a specific user and all its orders $result = $db->from('users') ->where('users.id')->is(2014) ->join('orders', function($join){ $join->on('users.id', 'orders.user_id'); }) ->delete(array('users', 'orders')); {% endhighlight %} {% highlight sql %} DELETE `users`, `orders` FROM `users` INNER JOIN `orders` ON `users`.`id` = `orders`.`user_id` WHERE `users`.`id` = 2014 {% endhighlight %}
