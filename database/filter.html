Filter records

Filtering records is one of the most important operation in database manipulation. Opis Database allows you to add filters to your queries by providing a set of methods that will make filtering records an easy task.

{% include subtitle.html anchor="Where" value="WHERE clause" %}

The WHERE clause is used to extract only those records that fulfill a specified criterion. Adding a WHERE filter to your query is done by calling the where, andWhere and orWhere methods.

{% highlight php startinline %} $result = $db->from('users') ->where('age', 18) ->select() ->all(); {% endhighlight %} {% highlight sql %} SELECT * FROM `users` WHERE `age` = 18 {% endhighlight %}
