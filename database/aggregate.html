Aggregate functions

Opis Database provides support for most common and widely used aggreate functions.

{% include subtitle.html anchor="Counting" value="Counting" %}

Counting records is done using the count method.

{% highlight php startinline %} $result = $db->from('users')->count(); echo 'There are ' . $result . ' users registred on this site.'; {% endhighlight %} {% highlight sql %} SELECT COUNT(*) FROM `users` {% endhighlight %}

Counting all values(NULL values will not be counted) of a column is done by passing the column's name as an argument to the count method.

{% highlight php startinline %} $result = $db->from('users')->count('description'); echo 'There are ' . $result . ' users that have provided a description for their profile.'; {% endhighlight %} {% highlight sql %} SELECT COUNT(`description`) FROM `users` {% endhighlight %}
