Fetch records

Fetching records from a database is done using the from method.

{% highlight php startinline %} $result = $db->from('users') ->select() ->all(); {% endhighlight %} {% highlight sql %} SELECT * FROM `users` {% endhighlight %}

If your query succeeds, the result will be an array of records, otherwise the method will return FALSE.

{% highlight php startinline %} if($result) { foreach($result as $user) { echo $user->name; } } else { echo 'No records were found'; } {% endhighlight %}

If you only want to retrive a single record, you can use the first method.

{% highlight php startinline %} $user = $db->from('users') ->select() ->first(); if($user) { echo $user->name; } else { echo 'No records were found'; } {% endhighlight %}
